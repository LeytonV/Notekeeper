Notekeeper is a notekeeping app created in Angular.

![2d19c72f09159da7411ab7823f2e69e6](https://github.com/LeytonV/Notekeeper/assets/40127046/7ea8ac06-a94f-4b15-b833-3d84f6775924)

Features:
- Pinning notes
- Archiving notes
- Trashing notes
- Recoloring notes
- Attaching images to notes
